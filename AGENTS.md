# cmt

CLI that generates conventional commit messages from staged git diffs using
configurable model providers.

## Essentials

- Package manager: bun
- Build/typecheck: `bun run build` (tsc)

```bash
bun install
bun run build
bun run start
bun test
bun run lint:fix
bun test:e2e
```
